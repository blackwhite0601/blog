CF1249E By Elevator or Stairs?

2021-10-02

这是一道动态规划的题目。

我们可以设 sis_isi​ 表示第 iii 层使用楼梯的最小花费，设 lil_ili​ 表示第 iii 层使用电梯的最小花费。

由题意也不难推出状态转移方程。

走楼梯：si=min(si−1,li−1)+si;s_i=min(s_{i-1},l_{i-1})+s_i;si​=min(si−1​,li−1​)+si​;

坐电梯：li=min(si−1+t,li−1)+li;l_i=min(s_{i-1}+t,l_{i-1})+l_i;li​=min(si−1​+t,li−1​)+li​;

最后输出第 iii 层的答案，即 sis_isi​ 与 lil_ili​ 的最小值。

#include<bits/stdc++.h>
#define N 200001
using namespace std;
int n,t,s[N],l[N],s[N],l[N];

int main(){
    l[1]=1e9;//第一层不能坐电梯
    scanf("%d%d",&n,&t);
    for(int i=1;i<n;i++)scanf("%d",&s[i+1]);
    for(int i=1;i<n;i++)scanf("%d",&l[i+1]);
    printf("0");
    for(int i=2;i<=n;i++){
        s[i]=min(s[i-1],l[i-1])+s[i];//走楼梯的情况
        l[i]=min(s[i-1]+t,l[i-1])+l[i];//坐电梯的情况，注意从楼梯转电梯需要加上等待时间t
        printf(" %d",min(s[i],l[i]));//输出最小值
    }
    return 0;
}
