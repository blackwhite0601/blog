CF490A Team Olympiad

2021-10-02 1 min read # CodeForces

这道题的思路很简单，开三个桶记录擅长不同技能的人的的编号，参赛队伍的最大数量就是三个桶中元素数量的最小值，最后输出参赛队员的编号即可AC。

上代码：

#include<bits/stdc++.h>
using namespace std;
int n,x,y,z,a[10001],b[10001],c[10001];
int main(){
    scanf("%d",&n);
    for(int i=1;i<=n;i++){
    	int t;
        scanf("%d",&t);
        if(t==1) a[++x]=i;
        else if(t==2) b[++y]=i;
        else c[++z]=i; 
    }
    int ans=min(x,min(y,z));
    printf("%d\n",ans);
    for(int i=1;i<=ans;i++) 
		printf("%d %d %d\n",a[i],b[i],c[i]);
	return 0;
}
