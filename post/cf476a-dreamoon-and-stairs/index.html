CF476A Dreamoon and Stairs

2021-10-02

通过题意可以发现，xxx 最大为 nnn （一步一步跳），最小为 $n / 2 + n % 2 $ （两步两步跳），可以直接枚举在这个范围内的 xxx ，找出 mmm 的倍数输出，代码如下：

#include<bits/stdc++.h>
using namespace std;
int n,m;
int main(){
	scanf("%d%d",&n,&m);
	for(int i=(n/2)+(n%2);i<=n;i++)
		if(i%m==0){
			printf("%d\n",i);
			return 0;
		}
	puts("-1");
	return 0;
}
