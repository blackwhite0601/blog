CF764B Timofey and cubes

2021-10-02

枚举样例后不难发现当 i mod 2=0i\bmod2=0imod2=0 时，aia_iai​的值被交换了两次，所以能得出以下规律：

ai=an−i+1a_i=a_{n-i+1}ai​=an−i+1​ (i mod 2=0)(i\bmod2=0)(imod2=0)

所以我们就可以只枚举 i mod 2=0i\bmod2=0imod2=0 的情况，得出以下代码：

#include<iostream>
using namespace std;
int n,a[200005];

int main() {
	scanf("%d",&n);
	for(int i=1; i<=n; i++)
		scanf("%d",&a[i]);
	for(int i=1; i<=n/2; i+=2)
		swap(a[i],a[n-i+1]);
	for(int i=1; i<=n; i++)
		printf("%d ",a[i]);
	return 0;
}
